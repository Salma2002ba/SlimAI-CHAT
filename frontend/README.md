# SlimAI Frontend (Vite + React)

This directory contains the SlimAI web frontend, generated initially from Google AI Studio / Gemini.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in `.env.local` (or environment) to your Gemini API key.
3. Run the app:
   `npm run dev`

All commands in this README are intended to be run from the `frontend/` directory.
